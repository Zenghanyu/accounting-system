=======================================
    📤 如何分享智能记账系统给好友
=======================================

方法1：复制整个文件夹
--------------------------
1. 选中整个 "accounting-system" 文件夹
2. 右键 → 发送到 → 压缩文件夹
3. 通过以下方式发送给好友：
   ✅ QQ/微信文件传输
   ✅ 邮件附件
   ✅ U盘拷贝
   ✅ 网盘分享（百度网盘、阿里云盘等）

好友收到后：
1. 解压文件夹
2. 双击 "直接打开.bat" 即可使用
3. 每个人的数据独立保存


方法2：上传到网盘分享
--------------------------
📦 百度网盘：
1. 登录 https://pan.baidu.com
2. 上传 accounting-system 文件夹
3. 右键文件夹 → 分享
4. 复制分享链接发给好友

📦 阿里云盘：
1. 登录 https://www.aliyundrive.com
2. 上传文件夹
3. 分享给好友

📦 微云：
1. 登录 https://www.weiyun.com
2. 上传并��享


方法3：发布到Gitee（国内GitHub）
--------------------------
如果GitHub访问不了，可以用Gitee：

1. 注册Gitee账号：https://gitee.com
2. 创建新仓库（选择公开）
3. 在当前文件夹右键 → Git Bash Here
4. 执行命令：
   git remote add gitee https://gitee.com/你的用户名/accounting-system.git
   git push gitee main

5. 好友访问：https://gitee.com/你的用户名/accounting-system
6. 点击"克隆/下载" → 下载ZIP


方法4：创建在线版本（需要部署）
--------------------------
如果想让好友直接访问网址：

A. 使用Vercel（国外）
   - 访问 https://vercel.com
   - 导入GitHub仓库
   - 获得网址分享

B. 使用Gitee Pages（国内，免费）
   1. 仓库推送到Gitee
   2. 进入仓库 → 服务 → Gitee Pages
   3. 启用服务
   4. 获得网址：https://你的用户名.gitee.io/accounting-system

C. 使用Netlify
   - 访问 https://netlify.com
   - 拖拽文件夹上传
   - 获得网址分享


=======================================
推荐方式：

✅ 近距离分享：U盘/直接复制
✅ 网络分享：网盘（百度网盘/阿里云盘）
✅ 技术用户：Gitee仓库
✅ 想要在线版：Gitee Pages
